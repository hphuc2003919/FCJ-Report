Trong kỳ thực tập với **chương trình First Cloud Journey (FCJ)**, tôi đã hoàn thành chương trình trong vòng **12 tuần**. Worklog ghi lại các hoạt động hàng tuần, trải nghiệm học tập và những ứng dụng thực hành với các dịch vụ AWS và các nhiệm vụ liên quan đến cloud.

**Tuần 1:** [Làm quen với AWS, Global Infrastructure, các Management Tools, Cost Optimization và hoàn thành các bài hands-on labs cơ bản.](1.1-week1/)

**Tuần 2:** [Nghiên cứu các khái niệm Networking cốt lõi của AWS như VPC architecture, security layers, hybrid connectivity và hoàn thành các bài lab liên quan.](1.2-week2/)

**Tuần 3:** [Brainstorm ý tưởng dự án, phân chia nhiệm vụ, lập kế hoạch workflow và thực hành các advanced networking labs như VPC Peering và AWS Transit Gateway.](1.3-week3/)

**Tuần 4:** [Phát triển frontend cho dự án đồng thời học các dịch vụ Compute và Storage như Amazon EC2, Auto Scaling và các bài hands-on labs về lưu trữ.](1.4-week4/)

**Tuần 5:** [Hoàn thiện integration giữa frontend–backend, dịch các AWS Blogs và thực hành với Amazon S3, Glacier, Snow Family, AWS Storage Gateway, AWS Backup, VM Import/Export và Amazon FSx.](1.5-week5/)

**Tuần 6:** [Tìm hiểu các dịch vụ Security và Identity như IAM, AWS Security Hub, Resource Tagging và tối ưu chi phí EC2 thông qua Lambda automation.](1.6-week6/)

**Tuần 7:** [Thực hành các advanced IAM features, Encryption bằng AWS KMS và ôn tập Secure Architectures để chuẩn bị bài thi giữa kỳ.](1.7-week7/)

**Tuần 8:** [Ôn tập Resilient Architecture, High-Performing Architecture, Cost-Optimized Architecture, thực hành các core AWS services và hoàn thành bài thi giữa kỳ.](1.8-week8/)

**Tuần 9:** [Ôn tập các AWS Database Services, thực hành RDS và NoSQL như DynamoDB, khám phá Data Lake Architecture và phân tích Cost & Performance.](1.9-week9/)

**Tuần 10:** [Làm việc với Amazon DynamoDB, xây dựng Serverless Data Lake, học Real-time Analytics, ETL Pipelines và Data Visualization bằng các dịch vụ AWS.](1.10-week10/)

**Tuần 11:** [Tham dự sự kiện AWS Cloud Mastery Series về DevOps, tìm hiểu Hybrid Cloud, Serverless Integration và thực hành Cost Optimization cho DevOps workloads.](1.11-week11/)

**Tuần 12:** [Ôn tập AWS Security Best Practices, triển khai Secure IAM Architectures và tham dự AWS Well-Architected Framework – Security Pillar workshop để nâng cao kiến thức thực hành về Cloud Security.](1.12-week12/)
